# 구조체
형태
```swift
/*
struct 구조체 이름 {
    프로퍼티와 메서드
}
*/
```

사용법
```swift
struct User {
    var nickname: String
    var age: Int

    func information() {
        print("\(nickname) \(age)")
    }
}

var user = User(nickname: "Cocoa", age: "20")

user.nickname // "Cocoa"
user.nickname = "Bubble"
user.nickname // "Cocoa"

user.information() // Cocoa 20
```
구조체를 생성하면 기본적으로 프로퍼티명 생성자가 만들어진다.
